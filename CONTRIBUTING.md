# Contributing

Truce is pre-1.0 and the contribution process is still being worked
out. 

If you're interested in contributing — a bug fix, a feature, or
becoming a regular maintainer — please [open an
issue](https://github.com/truce-audio/truce/issues/new) describing
what you have in mind. Drive-by PRs are welcome, but no promises.

The development workflow rules and release process live in
[`dev/docs/DEVELOPMENT.md`](dev/docs/DEVELOPMENT.md).
