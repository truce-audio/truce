# truce-xtask

Build, bundle, and install truce audio plugins.

Provides the core logic behind `cargo truce`. Handles building plugin binaries,
bundling them into platform-specific formats, code signing, installation to
system plugin directories, and validation.

## Commands

`install`, `build`, `run`, `new`, `test`, `status`, `clean`, `nuke`,
`validate`, `doctor`, `log`

Part of [truce](https://github.com/truce-audio/truce).
